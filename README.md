# NeV 設置場所見取図 要件判定チェックツール

NeV補助金（次世代自動車充電インフラ整備促進事業）の申請に必要な**設置場所見取図**が、補助金要件を満たしているかをAIで自動チェックするWebツールです。

## 機能

- **基礎充電 / 目的地充電** の2タイプに対応
- Google Gemini API による高精度な図面読み取り
- 補助金要件に基づく13項目のチェック
- 合格/不合格/要確認の3段階判定
- 結果のコピー機能

## チェック項目

### 共通項目
| # | チェック内容 |
|---|------------|
| 1 | 設置場所の記載 |
| 2 | 図面名称「設置場所見取図」の記載 |
| 3 | 作成者の記載 |
| 4 | 縮尺の記載 |
| 5 | 作成日の記載 |
| 6 | 道路の記載 |
| 7 | 出入口の記載 |
| 8 | 充電/充放電スペースの図示 |
| 9 | 敷地全体の形状把握 |
| 10 | 周辺施設との位置関係 |

### 基礎充電 固有項目
- 収容台数の記載
- 建物名称の明記
- 充電スペース数の記載

### 目的地充電 固有項目
- 道路名称の記載
- 施設名称の明記
- 充電スペース数の記載
- 案内板情報の記載（任意）
- 既設充電スペースの表示（任意）

## 使い方

1. [Google AI Studio](https://aistudio.google.com/apikey) で Gemini API キーを取得
2. ツールにAPIキーを入力
3. 「基礎充電」または「目的地充電」を選択
4. チェック対象のPDFをアップロード
5. 「要件チェックを実行」をクリック

## GitHub Pages での公開

1. このリポジトリをGitHubにプッシュ
2. Settings > Pages > Source を `main` ブランチの `/docs` フォルダに設定
3. 公開URLが発行されます

## 技術構成

- HTML / CSS / JavaScript（フレームワーク不使用）
- PDF.js（PDFレンダリング）
- Google Gemini API（図面解析）
- GitHub Pages（ホスティング）

## 注意事項

- 本ツールはAI判定のため、最終確認は必ず目視で行ってください
- Gemini APIキーは各自で取得が必要です（無料枠あり）
- PDFファイルは最大20MBまで対応
