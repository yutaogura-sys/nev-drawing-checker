# NeV 設置場所見取図 要件判定チェックツール

NeV補助金（次世代自動車充電インフラ整備促進事業）の申請に必要な**設置場所見取図**が、補助金要件を満たしているかをAIで自動チェックするWebツールです。

## 機能

- **基礎充電 / 目的地充電** の2タイプに対応
- Google Gemini API による高精度な図面読み取り
- 補助金要件（5-9-1 設置場所見取図）に基づくチェック
- 合格/不合格/要確認の3段階判定
- 結果のコピー機能

## チェック項目

### 共通項目（①図面基本情報 + ③入口 + ⑤充電スペース）
| # | チェック内容 |
|---|------------|
| 1 | 設置場所の記載（略称不可） |
| 2 | 図面名称「設置場所見取図」の記載 |
| 3 | 作成者の記載 |
| 4 | 縮尺の記載 |
| 5 | 作成日の記載 |
| 6 | 充電設備設置場所の入口 |
| 7 | 充電スペースの図示 |
| 8 | 施設全体の敷地形状 |
| 9 | 公道と他施設との位置関係 |

### 基礎充電 固有項目
- 建物名称の明記
- 収容台数の記載
- 充電スペース数の記載
- 既設充電スペースの表示（該当時）

> 基礎充電（マンション・集合住宅）では、補助金要件上 **公道の記載は不要** のためチェック対象外です。

### 目的地充電 固有項目
- **②公道（必須）**
  - 公道の記載
  - 公道名の記載（国道○号線、県道○号、市道○号線等）
- 施設名称の明記
- 充電スペース数の記載
- **④案内板（必須）**
  - 設置位置の図示
  - 向きの確認（両面→垂直、片面→平行）
  - 設置方法・仕様（ポール/壁付、片面/両面、サイズ）
  - 新設/既設の区別
- 既設充電スペースの表示（該当時）

## 使い方

1. [Google AI Studio](https://aistudio.google.com/apikey) で Gemini API キーを取得
2. ツールにAPIキーを入力
3. 「基礎充電」または「目的地充電」を選択
4. チェック対象のPDFをアップロード
5. 「要件チェックを実行」をクリック

## GitHub Pages での公開

1. このリポジトリをGitHubにプッシュ
2. Settings > Pages > Source を `main` ブランチの `/docs` フォルダに設定
3. 公開URLが発行されます

## 技術構成

- HTML / CSS / JavaScript（フレームワーク不使用）
- PDF.js（PDFレンダリング）
- Google Gemini API（図面解析）
- GitHub Pages（ホスティング）

## 注意事項

- 本ツールはAI判定のため、最終確認は必ず目視で行ってください
- Gemini APIキーは各自で取得が必要です（無料枠あり）
- PDFファイルは最大20MBまで対応
